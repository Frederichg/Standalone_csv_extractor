Project Excel_merger_ID

Context:

We have lots of .xlsx files that are generated by a behavior analysis system. All the files has the same basic structure but there is many different type. So the logic is constant but the details are changing.

I have many student whom are not familiar with programming so I do not what them to risk doing the merging wrong but I do not what to do all those merging my self. So the end result I want my student to be able to run this on their own computer (pc or mac) with a very simple deployment. So if possible, no need to install python on their machine only to install this program with everything needed included in the package. There will be a simple GUI that I will explain later (basic TKinker)). For a first version, we do not need to have the .exe and packaging, I just mention it so you can use toolbox that are easier to parkage and require no or few dependencies.  The progam will be in Python.

The context is we have files with many tests from our animals. One animal can have 100 tests and we can have 100 animals. So there is so many lines in the excel files. Somethime  we need to add a column to say for example, the Sex of the animal. We have another excel file that contain the unique animal ID number and a second column with the 'sex'. So instead of going manually, we would like a program that will look at the animal number for the 1000 lines file and go see on the ID_Sex.xlsx, look at the column with animal ID number, find the number that we look for then get the 'M or F' on the column called Sex. Here it is just an example, the names of files and colomn may vary. 

We need a single GUI that will be separated in left and right. In Left side is the source and Right is the destination. There will be on each side a 'browser' button to find the excel file that we are working with. Under the browsers there a 'sheet' dropdown selector, because our excel file has many sheet with variable names. Side note, it can be the same file as source and destination, sometime we just need to take from one sheet to another. Under the 'sheet' dropdown, there is another dropdown to select the 'common identifier'. In the simplest case it will be a column with the animal ID number on each side. Under this dropmenu, something to identify which column we what to copy and on the other side where (wich colum) we want to paste. Sometime we would like to copy-paste all column from the source to the recipient, so you may put a check box 'import all columns'
